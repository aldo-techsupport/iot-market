## 📊 Sensor Support

API ini mendukung **53 sensor** dalam **8 kategori**:

- Environmental (11 sensor)
- Electrical (11 sensor)
- Flow & Pressure (9 sensor)
- Machine Status (7 sensor)
- Safety (5 sensor)
- Agriculture & Water (5 sensor)
- Weather (3 sensor)
- Vibration & Rotation (2 sensor)

Lihat daftar lengkap di [API_IOT_DOCUMENTATION.md](API_IOT_DOCUMENTATION.md)

## 🔑 Key Features

1. **Sensor Filtering** - Pilih sensor yang diaktifkan per device
2. **Device Management** - API Key unik per device
3. **Flexible Input** - Kirim sensor apa saja, backend yang filter
4. **Statistics** - Agregasi data (avg, min, max)
5. **Batch Insert** - Kirim multiple data sekaligus
